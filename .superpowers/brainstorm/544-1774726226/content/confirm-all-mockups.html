Approved Decisions Summary:
  • Shell structure: Dark left nav (#1e293b) + Light top bar (#fff)
  • Context switching: Full nav replace with project switcher dropdown
  • Nav collapsibility: Collapsible from v1, default expanded (200px → 52px)
  • Page header: Spacious — title + summary stats, then filters + actions row
  • Breadcrumb: Top bar, clickable segments
  • Create button: Context-aware single action in top bar
  • Notifications: Bell icon with dropdown + badge
  • Profile menu: Avatar with dropdown (My Profile, Prefs, Settings, Help, Sign Out)
  • Accent color: Amber (#f59e0b) for logo, active indicators, highlights
